Store & Inventory Settings
Refresh Shop Only When Cleared: If active, NeoBuyer+ will only buy items after a shop clear and it will not continuously refresh or navigate to other shops. This is useful if you want to only buy items in shop clears to minimize the possibility of long restock bans.

This feature is also useful if you have multiple shop tabs open and you just want to restock when shops are empty and wait for the next shop clear.

Activate this option while also setting "Store Numbers to Cycle Through When Stocked:" to [].
Store Numbers to Cycle Through When Stocked: If a store in the list is stocked, NeoBuyer+ will wait the specified number of seconds before moving to the next shop.

It cycles through multiple shops until finding one that's not stocked.

Go for 2nd Best Item: Better for slower internet connections.

Instead of aiming for the best item in stock, NeoBuyer+ prioritizes the second-best item for a better chance of success.

It will still buy the sole worthwhile item if only one is available.
Auto-Click Confirm Window: Enables NeoBuyer+ to Auto-Click the Confirm Window Buttons.
Bypass Confirm Window: If active, NeoBuyer+ will bypass the confirm pop-up Neopets generates, redirecting the user straight to the haggling page.

Although, inoffensive, Neopets can possibly detect the use of NeoBuyer+ through this feature. Activate this option at your own risk.
