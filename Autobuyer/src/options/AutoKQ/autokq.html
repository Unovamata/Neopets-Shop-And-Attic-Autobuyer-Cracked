Maximum Price to Insta-Buy an Item: When the price of an item falls below this specified amount in a user's shop, the AutoKQ process will purchase that item without any further searches, minimizing the risk of Shop Wizard bans.

If the AutoKQ process cannot find a price below this threshold, it will continue searching until an item's price is less than the specified amount or the number of resubmits per item has been reached. The SW AutoBuyer will always buy the cheapest item out of all searches.

If you want to disable this feature, set this value to 0. 		NPs
