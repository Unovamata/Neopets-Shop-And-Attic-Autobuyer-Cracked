Export & Load Settings
Always verify and test if your presets have been exported as expected!
Refresh the page if not and export your setting presets again.
Export Store Numbers to Cycle: If you enable this option, the configuration exporter will include the 'Store Numbers to Cycle Through When Stocked' data.

This option is only included if you are concerned about sharing this information, as users may be able to match your shop's stock with the stores you browse.

This data may be considered potentially identifiable.
Export Buyable Items List: If you enable this option, the configuration exporter will include your 'Restock List' data.

This is not identifiable information.
Export Shop Stock: If you enable this option, the configuration exporter will include your entire 'Shop Inventory' data.

This is HIGHLY identifiable information.

You could potentially get frozen if you share this!

Please proceed with caution.
Export Shop Sales History: If you enable this option, the configuration exporter will include your entire 'Shop Sales History' data.

This is HIGHLY identifiable information.

You could potentially get frozen if you share this!

Please proceed with caution.
Export Blacklists: If you enable this option, the configuration exporter will include your 'Blacklist' data.

This is not identifiable information.
Export Neopets Security PIN: If you enable this option, the configuration exporter will include your Neopets' account PIN.

Only share this information between local sessions of the extension if possible.

This is HIGHLY identifiable information.

You could potentially get frozen if you share this!

Please proceed with caution.
Export Attic Last Refresh Time: If you enable this option, the configuration exporter will include the 'Last Restock Time in NST' data.

This option is only included if you are concerned about sharing this information, as users may be able to match your account based on Trading Post lots.

This data may be considered potentially identifiable.
Export Purchased Items History: If you enable this option, the configuration exporter will include your entire 'Purchase History'.

This is HIGHLY identifiable information as it includes your username and purchase dates.

You could potentially get frozen if you share this!

Please proceed with caution.
Export AutoKQ Logs: If you enable this option, the configuration exporter will include your entire 'AutoKQ Log'.

This is HIGHLY identifiable information as it includes your username and some pet names.

You could potentially get frozen if you share this!

Please proceed with caution.
Export Extension's Emails: If you enable this option, the configuration exporter will include your entire Email list.

To be clear, this is NOT your email address or any of your emails in your inbox from your personal email, these are the extension's propietary Emails and nothing else.

This is not identifiable information.
