Final 0.7.26.39v Version Release

Version 0.7 Completed:

    Thank you for your patience and support during the development of version 0.7v. I am happy to announce that NeoBuyer+ version 0.7v is finally complete. I can now start working on the highly anticipated version 0.8v.

  • Show Your Support: Consider Starring NeoBuyer+ or Donating as a token of appreciation. This is made free for everyone to use after hundreds of hours of hard work, so thank you for your continuous support.


  • Discord Server: Connect with other users of NeoBuyer+, Join Now!

    • Discord Server Purpose: The new NeoBuyer+ Discord server is available to the public. You can connect with other NeoBuyer+ users by sharing configuration presets, tips, and tricks. Moreover, you can suggest features and follow the project's development in real time. Have fun with us!

      In this server, I will also promote my upcoming game so I hope you follow the development of it closely.


  • Changes & Optimizations:

    • AutoAttic State: This update got severely delayed because of the state of the AutoAttic. While trying to improve its code, the system broke for a long time since this feature had leftover code from the original developer of the extension. The AutoAttic bot works now. However, bugs may appear while utilizing this bot. More information on this later.

    • User Manual: By popular demand, I have released a User Manual in version 0.7.26.39v by clicking the "User Manual" button the extension will redirect you to its respective page in the User Manual. Read NeoBuyer+ Manual.

    • Better Toolbar: Reorganized the Toolbar again in preparation for several features in version 0.8v. Now you can find all Github-related information in its respective submenu.

    • Bug Fixes and Extra Features:
      • Update Checker Optimized.
      • Fixed some bugs in the AutoAttic, AutoPricer, and AutoKQ processes.
      • Added some new graphs in the AutoAttic History menu.
      • NeoPass announcement text entries were removed from the History database. Thank you for this change Chrisli-03!


  • What is planned for version 0.8?

    • Rebranding: We will rebrand NeoBuyer+ to AutoBuyer+ and eventually release it to the Chrome Extensions store. AutoBuyer+ has come a very far way, and it's all thanks to your unwavering support.

    • Real-time database updates: This has been a very sought-after feature since the conception of NeoBuyer+.

    • AutoQuickStock: Another very sought-after feature to automatically sort your inventory while AutoBuying.

    • AutoAttic+: A new, faster, and more intelligent algorithm to handle all your AutoAttic needs built from the ground up.

    • AutoBuyer+: A new, faster, and more intelligent algorithm for AutoBuying in Neopia's main shops. Including day-of-the-week shop visit configuration, per-shop configurations, per-shop lists, per-shop visit duration instead of linearly cycling shops, Real Shop Clear/Ban Detection, and more.

    • Auto Daily Quest Solver: Never lose a weekly prize again.

    • AutoBattledome: For all your AutoBattledome needs! I hope you utilize this function along with the AutoQuickStock feature.

    • Brand New GUI: Before, the systems NeoBuyer+ had were an obstacle because I did not know as much about extension development. Not anymore! I am confident you will love it.

    • A New Mascot Assistant!? May I present you, BAI-kun.

      • BAI-kun is an interactive mascot assistant similar to a Tamagotchi that will help you understand AutoBuyer+ thoroughly.
      • You can interact with them, play with them, learn new things about the extension, and buy them goodies with "+ Coins". You can obtain "+ Coins" by using AutoBuyer+ and interacting with BAI-kun (+ Coins are for entertainment only and do not represent real currency).
      • BAI-kun will adapt and react to your AutoBuyer+ usage while offering useful advice to optimize your AutoBuyer+ usage. Please treat them nicely BAI-kun may hold a grudge if you are mean to them.
      • (Tentative design/concept subject to change).


    • And that is it! For now... Version 0.8v is a HUGE update. I may add more features as time progresses, but this is the new roadmap and direction NeoBuyer+ will take. I hope you continue to support the extension for the foreseeable future.

    Thank you for using NeoBuyer+ it was a pleasure! Let us pave the way for AutoBuyer+!
