Almost Abandoned Attic Autobuyer Enabled: Enables Almost Abandoned Attic Auto-Buyer.
Refresh Settings
Should Refresh Automatically: This enables automatic refreshing for the attic buyer.

You can disable it to manually refresh the page.
Last Restock Time in NST: This is the last known restock time for the attic in Neopian Standard Time (NST).
Visit the Attic Antics Neoboard to find this timestamp.

This helps the NeoBuyer+ refresh more strategically, reducing refresh bans.

Go for 2nd best item: This is useful if you have a slow internet connection. Instead of trying to get the best item in stock, the autobuyer will try to get the 2nd-best item, so that you have a higher chance of success. If there is only 1 item in stock worth buying, it will still try to buy that one item.
Attempt to Refresh Through Page Load Failures: When the page fails to load because Neopet's server rejected the request, which happens sometimes, this setting will refresh the page anyways instead of stopping on the failed page.

Use a Blacklist of Items to not Purchase: NeoBuyer+ will be prevented from clicking on specific items inside the blacklist, even if they meet the profit criteria.

Visual Settings
Show Status Banner: This setting toggles the banner text shown on the shop pages.

Although, inoffensive, Neopets can possibly detect the use of NeoBuyer+ through this feature. Activate this option at your own risk.
Highlight Desired Attic Items: Neobuyer+ highlights items of interest:

- The highest valued item in stock is highlighted in orange.
- Other items of interest will be highlighted in green.

Although, inoffensive, Neopets can possibly detect the use of NeoBuyer+ through this feature. Activate this option at your own risk.
Alert Settings
Show Chrome Notifications: Chrome notifications will appear when important events occur.
Change Page Information: If enabled, NeoBuyer+ will change the titles on pages to reflect the state of the AutoBuyer at any given point and time.

Although, inoffensive, Neopets can possibly detect the use of NeoBuyer+ through this feature. Activate this option at your own risk.
Sound Alerts: The AutoBuyer provides audible notifications during page actions.
