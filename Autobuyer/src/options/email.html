NeoBuyer Email Setup

Start by creating a free EmailJS account at www.emailjs.com. This is a third party service which will allow you to send emails automatically to yourself. Then create an "Email Service" by linking an email account that you own. Once you have done that, there will be a Service ID listed:



Then, create a new email template will the following contents. You can create a link to automatically link to the JellyNeo listing for the item by linking to the URL "https://items.jellyneo.net/search/?name={{item}}". You can style the email however you want to, but the {{status}} {{item}} {{notes}} fields are mandatory.



You will now have a Template ID visible after saving:



Lastly, you can view your User ID:



These are the 3 IDs that you will need to add into your options page for NeoBuyer. You can purcahse an item to test that everything is set up correctly, and you should recieve an email if NeoBuyer is running.
